----

**tClip** is a simple clipboard manager written in core [Tcl/Tk](https://www.tcl.tk).

----

### Features

* On Top

* Tk Themes

* Editable Text

* Add custom Clips

* Command Line Support

* Runs in GNU/Linux, MS-Windows and Unix (MacOS not tested)
